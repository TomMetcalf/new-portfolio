Workout Tracker

This fitness app is a simple workout tracker, where users can log a workout, with a title, load and number of reps. Logged workouts can be edited and deleted. All fields are required and users are alerted when trying to submit a workout with missing fields. Workouts are stored in a MongoDB database and a loading animation is shown whilst the data is retrieved from the database.

Users can sign up and log in and only their own workouts will be displayed. A calendar can also be used to display workouts for a particular date.

NC News

The aim for this project was to build a news website, where users can view articles, vote and comment on them, filter them by topic, sort them by date published, comment or vote count and order them in either ascending or desending order.

The backend API for this site was built using SQL and the frontend was created using React.
